Kitaev Chain (Topological Superconductor)

This example computes certified lower bounds on the ground-state energy of the Kitaev chain ‚ÄĒ the simplest model that hosts a topological superconducting phase with Majorana zero modes at its ends.

If you haven't seen fermionic operators before, start with the Fermionic Ground State (XY Model) example, which builds the language of creation/annihilation operators and the CAR from scratch. Here we assume that background and focus on what's new: pairing terms that break particle-number conservation, Majorana operators, the topological vs. trivial phase, and the difference between open and periodic boundary conditions.

The Kitaev chain Hamiltonian

Picture $N$ sites on an open wire (no ring). Each site can hold zero or one spinless fermion. The Hamiltonian has three terms:

\[H = -\mu \sum_{i=1}^{N} a_i^\dagger a_i \;-\; t \sum_{i=1}^{N-1}\bigl(a_i^\dagger a_{i+1} + a_{i+1}^\dagger a_i\bigr) \;+\; \Delta \sum_{i=1}^{N-1}\bigl(a_i\, a_{i+1} + a_{i+1}^\dagger a_i^\dagger\bigr).\]

TermPhysicsEffect on particle number
$-\mu\, a_i^\dagger a_i$Chemical potential ‚ÄĒ energy cost of occupying a siteConserves $N_{\mathrm{tot}}$
$-t\,(a_i^\dagger a_{i+1} + \text{h.c.})$Hopping ‚ÄĒ fermion tunnels between neighboursConserves $N_{\mathrm{tot}}$
$\Delta\,(a_i\, a_{i+1} + \text{h.c.})$Pairing ‚ÄĒ creates or destroys fermion pairs from a condensateBreaks $N_{\mathrm{tot}}$ conservation

The pairing term is the key novelty compared to the XY model. It models a p-wave superconductor: the pairing amplitude connects neighbouring sites (odd under spatial inversion in momentum space). Because pairs of fermions are created and destroyed, particle number $N_{\mathrm{tot}}$ is no longer conserved. What is conserved is fermionic parity: $P = (-1)^{N_{\mathrm{tot}}}$. Every term in $H$ changes the fermion count by $0$ or $\pm 2$, so even/odd parity never flips.

Open boundary conditions ‚ÄĒ no parity constraint needed

The XY model example uses periodic boundaries (a ring), where the boundary term picks up a parity operator $P$ and we must impose $P = I$ as a constraint. Here we use open boundaries (a wire) ‚ÄĒ there is no boundary twist, so no parity constraint is required. The SDP relaxation searches over all parity sectors automatically.

Majorana operators and the topological sweet spot

Every complex fermion can be split into two Majorana (real) fermions:

\[c_{2j-1} = a_j + a_j^\dagger, \qquad c_{2j} = \frac{a_j - a_j^\dagger}{i}.\]

These are self-conjugate ($c_l^\dagger = c_l$) and obey $\{c_l, c_m\} = 2\delta_{lm}$. Think of them as splitting a complex number into its real and imaginary parts ‚ÄĒ except the "parts" are operators that anticommute.

The sweet spot: $\mu = 0$, $t = \Delta$

Set $\mu = 0$ and $t = \Delta$. The Hamiltonian simplifies to:

\[H = it \sum_{j=1}^{N-1} c_{2j}\, c_{2j+1}.\]

Each term pairs the Majorana $c_{2j}$ from site $j$ with $c_{2j+1}$ from site $j{+}1$ ‚ÄĒ the pairing reaches across sites. But the first Majorana $c_1$ and the last $c_{2N}$ appear in no term of $H$: they are the unpaired Majorana zero modes. Any operator that commutes with $H$ and is built from $c_1$ and $c_{2N}$ alone costs zero energy ‚ÄĒ hence "zero mode." These edge modes are the hallmark of the topological phase.

Under the Jordan‚ÄďWigner transformation, $ic_{2j}\,c_{2j+1} = X_j X_{j+1}$, so the sweet-spot Hamiltonian maps to the open XX chain:

\[H_{\text{sweet}} = \sum_{j=1}^{N-1} X_j X_{j+1}, \qquad E_0 = -(N-1).\]

We'll verify this Majorana‚ÜĒPauli connection explicitly in the bonus section below.

Exact diagonalisation oracle

To check our SDP bounds we need an exact answer. We build the full $2^N \times 2^N$ Hamiltonian matrix via the Jordan‚ÄďWigner mapping (fermion operators ‚Üí tensor products of Pauli matrices) and take the smallest eigenvalue. This is brute-force exact diagonalisation ‚ÄĒ it works for small $N$ and serves as our verification oracle.

using LinearAlgebra

# Jordan‚ÄďWigner building blocks
const PAULI_Z  = ComplexF64[1 0; 0 -1]
const PAULI_I  = Matrix{ComplexF64}(I, 2, 2)
const JW_RAISE = ComplexF64[0 1; 0 0]   # ŌÉ‚Āļ = |0‚ü©‚ü®1|
const JW_LOWER = ComplexF64[0 0; 1 0]   # ŌÉ‚ĀĽ = |1‚ü©‚ü®0|

"""
    jw_fermion_op(mode, dagger, nmodes)

Jordan‚ÄďWigner representation of a fermionic operator on `nmodes` sites.
Returns the `2^nmodes √ó 2^nmodes` matrix for `a‚Ä†_mode` (if `dagger=true`)
or `a_mode` (if `dagger=false`).
"""
function jw_fermion_op(mode::Int, dagger::Bool, nmodes::Int)
    mats = [site < mode ? PAULI_Z :
            site == mode ? (dagger ? JW_RAISE : JW_LOWER) :
            PAULI_I for site in 1:nmodes]
    return reduce(kron, mats)
end

"""
    kitaev_chain_exact(nsites; őľ=0.0, t=1.0, őĒ=1.0)

Exact ground-state energy of the `nsites`-site Kitaev chain via full
exact diagonalisation.
"""
function kitaev_chain_exact(nsites::Int; őľ::Real = 0.0, t::Real = 1.0, őĒ::Real = 1.0)
    a  = [jw_fermion_op(i, false, nsites) for i in 1:nsites]
    a_dag = [jw_fermion_op(i, true,  nsites) for i in 1:nsites]
    dim = 2^nsites
    H = zeros(ComplexF64, dim, dim)
    for i in 1:nsites                                       # chemical potential
        H .-= őľ .* (a_dag[i] * a[i])
    end
    for i in 1:(nsites - 1)
        H .-= t .* (a_dag[i] * a[i+1] + a_dag[i+1] * a[i])  # hopping
        H .+= őĒ .* (a[i] * a[i+1] + a_dag[i+1] * a_dag[i])  # pairing
    end
    return eigmin(Hermitian(H))
end

Solving with NCTSSoS.jl

We now reproduce these exact values using semidefinite programming relaxations. The solver doesn't know the answer ‚ÄĒ it searches for the lowest expectation value of $H$ consistent with the fermionic algebra (CAR). The result is a certified lower bound on the true $E_0$.

SDP Solver

These examples use Mosek via MosekTools. Any SDP-capable solver works: replace Mosek.Optimizer with COSMO.Optimizer or Clarabel.Optimizer for open-source alternatives.

    "MSK_IPAR_LOG" => 0,

Phase diagram

The Kitaev chain has a sharp topological phase transition controlled by the ratio $|\mu|/(2|t|)$:

RegimePhaseEdge modes?
$\mu< 2
$\mu> 2
$\mu= 2
$\Delta = 0$ (any $\mu$, $t$)Trivial (metal)No ‚ÄĒ no superconductivity at all

Our four cases sample each regime. Case 1 sits at the sweet spot deep in the topological phase; Case 2 is well into the trivial phase; Case 3 is topological but not at the sweet spot; Case 4 has no pairing, so no superconductivity and no topology.

Why topological matters

In the topological phase, the two Majorana zero modes ($c_1$ and $c_{2N}$) encode a single fermionic degree of freedom that is delocalised across the entire chain. This nonlocal storage makes the qubit robust against local perturbations ‚ÄĒ the foundation of proposals for topological quantum computing with Majorana-based qubits.

Bonus: Sweet-spot via PauliAlgebra

We showed above that at $\mu = 0$, $t = \Delta$, the Majorana form of the Kitaev chain maps under Jordan‚ÄďWigner to the open XX chain: $H = \sum_{j=1}^{N-1} X_j X_{j+1}$. Let's verify this directly using the create_pauli_variables interface.

Summary

The physics in one paragraph

The Kitaev chain is a 1D chain of spinless fermions with nearest-neighbour hopping and p-wave pairing. The pairing term breaks particle-number conservation but preserves fermionic parity. When $|\mu| < 2|t|$ and $\Delta \neq 0$, the chain is in a topological superconducting phase with unpaired Majorana zero modes at its ends. At the sweet spot ($\mu = 0$, $t = \Delta$), the Majorana representation makes the physics transparent: the Hamiltonian pairs Majoranas across bonds, leaving two dangling zero modes. Our SDP relaxation reproduces the exact ground-state energy across all four parameter regimes.
